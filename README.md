## DrawStarts

## 目的
    这个项目主要目的是记录下一下可能会使用到的前端相关知识，进行学习和练习。

## 常用命令

``` bash
# 安装依赖
npm install

# 启动服务
npm run dev || npm run serve

# 启动后台服务
npm run db:serve

# 打包
npm run build

```
